\\ Final Project CIS 25
\\ Project SS is a rudimentary game which uses shapes and colors to model characters, enemies, and one type of projectile
\\ To shoot one can use the A,D and Space keys to fire left, right, and up respectively
\\ The game ends if player health reaches 0 or if the player defeats enough enemies
\\ Further development/easter eggs may be added in the future
\\ The program functions using a combination of structures, for loops, while loops, and a great deal of sfml3 functions all communicating between the three files header.h, funcs.cpp, and main.cpp in order to hand different events/inputs
\\ Player starts with 250 health
\\ Enemies deal 10 damage when colliding with player
\\ Player wins by defeating 25 enemies
